## About this note

這是基於 Github 上 type-challenges 的內容做出的 answer blog.

記錄自己解出來的每個 challenge 內容，並加上與其他工程師們討論後得出的結論。

由於我本身是設計師出身，整篇筆記都是用淺顯易懂的方式，讓非工程師的人也能明白原理。希望這個 Blog 能對在 Typescript 學習之路上的大家有所幫助。

cheers!
